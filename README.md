# 📱 Indian Mobile Market Analysis

A complete end-to-end data analysis project on Indian mobile phone 
market using Python, SQL Server, and Power BI.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL Server (SSMS)
- Microsoft Power BI
- Microsoft Excel
- Jupyter Notebook

---

## 📁 Dataset
- Source: Indian Mobile Phone Specifications & Pricing Dataset
- Records: 15,000+ mobile phones
- Features: Brand, Price, RAM, Battery, Device Type and more

---

## 🔍 What I Analyzed
- Average price by brand
- Samsung vs Apple vs other brands comparison
- Budget vs premium phone distribution
- Battery and RAM trends across price ranges
- Price filtering and segmentation

---

## 📊 Project Steps
1. Data loading and exploration with Pandas
2. Data cleaning (handling nulls, renaming columns, filtering)
3. SQL queries for aggregation and grouping
4. Excel for quick summaries and pivot tables
5. Power BI dashboard for final visualization

---

## 💡 Key Findings
- Samsung has the highest number of models in Indian market
- Budget phones (under ₹15,000) dominate the market
- Higher RAM strongly correlates with higher price
